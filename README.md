## Featured Projects

### Anatomy-informed Synthetic Pre-training for Medical Segmentation

Research project related to **MICCAI 2026**, focusing on synthetic supervised pre-training for medical image segmentation.  
The project explores how anatomical priors, spatial anchors, and topology-aware constraints can generate anatomically plausible synthetic image-label pairs for downstream segmentation models.

**view preprint：https://arxiv.org/abs/2603.00979**

**Repository:** `fdsl`  
**Keywords:** Medical Segmentation · Synthetic Data · UNETR · SwinUNETR · MICCAI

---

### Microfluidic Detection Platform for Pathogens and Resistance Genes

A modular centrifugal microfluidic chip system for rapid multi-target detection of sepsis-related pathogens and antimicrobial resistance genes.  
The project integrates chip structure design, fluid simulation, experimental validation, and AI-assisted fluorescence curve interpretation.

**Repository:** `ai_microfluidics`  
**Keywords:** Microfluidics · LAMP · Pathogen Detection · COMSOL · SolidWorks · AI Interpretation

---

### Wearable 12-lead ECG Monitoring System

A wearable ECG monitoring project based on liquid-metal smart textiles and deep learning.  
The algorithm pipeline includes ECG preprocessing, representation learning, supervised contrastive learning, prototype-based distance modeling, and open-world abnormality recognition.

**Repository:** `Ecg-detection`  
**Keywords:** ECG · Wearable Sensing · ResNet1D · Open-world Recognition · Medical Signal Processing

---

### AI-assisted Cr(VI) Fluorescence Detection

An intelligent visual detection platform for Cr(VI) quantification based on carbon-dot fluorescence sensing and machine learning.  
The system combines fluorescence data acquisition, RGB feature extraction, k-means-based color-region processing, model training, and visualization.

**Repository:** `Cr-detection`  
**Keywords:** Cr(VI) Detection · Fluorescence Sensing · Machine Learning · RGB Analysis · Python GUI

---

### fMRI-based Cross-modal Face Reconstruction

A visual reconstruction project using fMRI signals to jointly decode identity, expression, and gender attributes.  
The framework combines feature disentanglement, brain-region selective linear mapping, and multi-conditional adversarial generation.

**Repository:** `MTDLN-for-f-mri-reconstruction`  
**Keywords:** fMRI · Visual Reconstruction · mcGAN · Feature Disentanglement · Medical Imaging

---

### YOLO Motion Detection

A computer vision project for object detection and motion analysis using YOLO-based models.  
The repository focuses on real-time detection, video/image processing, and visual output generation.

**Repository:** `yolo_motion_detect`  
**Keywords:** YOLO · Object Detection · Motion Detection · OpenCV · Python

### Lung Tumor Segmentation Task

**Repository:** `UM-Madison-task`  
**Keywords:** Lung Tumor Segmentation · Medical Imaging · Deep Learning · Jupyter Notebook
